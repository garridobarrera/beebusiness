Servicios REST para profesionales

Catálogo de servicios disponibles para realizar operaciones de profesionales

# Recurso Descripción ¿Autenticación?
1 [POST] /rest/profesional/authenticate Obtiene el token de autenticación a partir de las credenciales de acceso (username/password). NO
2 [GET] /rest/profesional Obtiene la información personal del profesional autenticado. SI
3 [GET] /rest/profesional/topics/{tipo} Obtiene los sectores/topics del profesional autenticado. SI
4 [GET] /rest/profesional/perfiles Obtiene los perfiles del profesional autenticado. SI
5 [GET] /rest/topics?tipo=value Obtiene los sectores/topics disponibles. NO
6 [GET] /rest/perfiles Obtiene los perfiles disponibles. NO
