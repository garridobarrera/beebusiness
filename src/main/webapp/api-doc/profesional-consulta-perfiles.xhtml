[GET] /rest/profesional/perfiles

Obtiene los perfiles del profesional autenticado para el token indicado.

Información.
  • HTTP Method: GET
  • Formato salida: JSON
  • Autenticación requerida: Sí
URL del recurso. http://IP:PORT/beebusiness/rest/profesional/perfiles
Parámetros headers.
token (*) token de autenticación.
Ejemplo de petición.

[GET] http://IP:PORT/beebusiness/rest/profesional/perfiles

HEADER: "token":"tcj4e69TclbQrmP4C6jR3N+WFKxbUvmSXl+NJw1A//B1AyYlLb5E771noIHRUdv6dak1lmotlnswoFtQqQSWJ4zWNnHr1htB"

OUTPUT: {{ id: 1 nombre: "Jefe" }}

Códigos de estado disponibles.
