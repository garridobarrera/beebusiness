[POST] /rest/profesional/authenticate

Obtiene el token de autenticación a partir de las credenciales de acceso (username/password).

Información.
  • HTTP Method: POST
  • Formato entrada: JSON
  • Formato salida: JSON
  • Autenticación requerida: No
URL del recurso. http://IP:PORT/beebusiness/rest/profesional/authenticate
Parámetros.
username (*) identificador de profesional
password (*) clave de profesional
Ejemplo de petición.

[POST] http://IP:PORT/beebusiness/rest/profesional/authenticate

INPUT: {"username":"XXX","password":"YYY"}

OUTPUT: {valid: true, value: "tcj4e69TclbQrmP4C6jR3N+WFKxbUvmSXl+NJw1A//B1AyYlLb5E771noIHRUdv6dak1lmotlnswoFtQqQSWJ4zWNnHr1htB"}

Códigos de estado disponibles.
200 OK Operación realizada correctamente
401 Unauthorized Credenciales incorrectas
