[GET] /rest/topics?tipo=value

Obtiene los sectores/topics.

Información.
  • HTTP Method: GET
  • Formato salida: JSON
  • Autenticación requerida: No
URL del recurso. http://IP:PORT/beebusiness/rest/topics?tipo=value
Parámetros Request.
tipo tipo de topic a consultar (posibles valores: "TEMATICA", "SECTOR")
Ejemplo de petición (1).

[GET] http://IP:PORT/beebusiness/rest/topics?tipo=SECTOR obtiene los topics por tipología SECTOR.

OUTPUT: {{ id: 1 , nombre: "Administración Pública", tipo:"SECTOR" }}

Ejemplo de petición (2).

[GET] http://IP:PORT/beebusiness/rest/topics?tipo=TEMATICA obtiene los topics por tipología TEMATICA.

OUTPUT: {{ id: 2 , nombre: "Formación", tipo:"TEMATICA" }}

Ejemplo de petición (3).

[GET] http://IP:PORT/beebusiness/rest/topics obtiene los topics sin filtrar por tipología.

OUTPUT: {{ id: 1 , nombre: "Administración Pública", tipo:"SECTOR" },{ id: 2 , nombre: "Formación", tipo:"TEMATICA" }}

Códigos de estado disponibles.
200 OK Operación realizada correctamente
